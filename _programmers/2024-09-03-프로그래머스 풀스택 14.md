### 🌊 자바스크립트 함수 4가지 종류

``` javascript
function add1(x, y) {
    return x + y
}
```
- 기본 형태<br>
<br>

``` javascript
let add2 = function(x,y) {
    return x + y
}
```
- 모듈화<br>
<br>

``` javascript
const add3 = (x,y) => {
    return x + y
} 
```
- function의 역할을 하는 =>(화살표 함수 arrow function)<br>
<br>

``` javascript
var add4 = (x,y) => x + y
```
